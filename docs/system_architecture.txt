# QueryLens System Architecture

QueryLens performs hybrid SQL performance analysis by combining static query inspection with runtime execution plan evidence.

## Pipeline

SQL Query
→ ANTLR Parser
→ AST Feature Extraction
→ Static Rule Detection
→ Execution Plan Analyzer
→ Correlation Engine
→ Evaluation Metrics

## Components

Static Analyzer
Detects performance anti-patterns using grammar-aware AST inspection.

Execution Plan Analyzer
Extracts operator-level runtime behavior from SQL Server execution plans.

Correlation Engine
Matches static warnings with supporting runtime evidence.

Evaluation Framework
Computes quantitative metrics for detection accuracy and validation effectiveness.
